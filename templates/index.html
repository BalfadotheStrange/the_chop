The database used by this site is publicly available here.
This site only uses the city of Denver's database.
The server uses the address entered above to locate the registered owner and the address the owner uses as a tax base for the building. It then uses that tax address to discover all other buildings registered to the same address. Then it shows that information in a list and on an interactive map.
